Frequently Asked Questions

What are credits?

Credits are the currency of Pretam.ai, used to power resource-intensive tasks such as generating app code, making API calls, and running complex AI agent workflows. One credit generally equates to a small, specific action, allowing us to manage usage fairly across all plans.

Can I switch plans?

Yes, you can easily switch between our Hobby and Pro plans at any time directly from your account dashboard. Your new plan and billing cycle will start immediately.

Is coding required?

No, Pretam is designed as a no-code platform. You can build powerful, functional applications and workflows using natural language descriptions alone. While the Pro plan offers advanced code agents for developers, coding is never required to get started.

Quick Start: Build a Simple Weather Agent
  1. Create a new project in your Pretam.ai dashboard.
  2. Describe your agent's purpose in the prompt: "I need an agent that fetches the current weather for a specified city and emails it to me."
  3. Pretam's Meta Agents will automatically scaffold the necessary tools (weather API, email integration).
  4. Follow the on-screen prompts to connect your email account.
  5. Test your agent and deploy it. You've just built your first AI-native workflow!

**Q: My API integration failed to authenticate. What should I do?**
